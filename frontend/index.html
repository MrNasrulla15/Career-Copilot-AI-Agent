              AI Stack Architecture

              Core technology integrations & Agent configuration

              Career Copilot is architected as an orchestrator pipeline running specialized sub-agents. It leverages the **Google Agent Development Kit (ADK)** to establish type-safe state transitions and coordinates document tools via **Model Context Protocol (MCP)**.

              Google ADK Framework Model Context Protocol (MCP) Gemini 2.5 LLM FastAPI Python Server Google Cloud Run Ready

              Specialist Agent Roster

              0. DocumentProcessorAgent

              Spawns MCP server subprocess via `McpToolset` to parse resumes using PDF/Docx text extractors.

              1. ResumeAnalysisAgent

              Extracts a structured 8-field JSON schema outlining skills, titles, and total years of candidate experience.

              2. JobAnalysisAgent

              Parses job listings into key requirements, seniority markers, and target technical stacks.

              3. GapAnalysisAgent

              Calculates skill overlaps, prioritize missing terminology, and scores overall match percentage.

              4. CareerStrategyAgent

              Draws timelines, maps immediate learning priorities, and recommends target side projects.

              5. InterviewPrepAgent

              Generates interview kits with behavioral, system design, and technical questions + STAR answer frameworks.

              ADK Sequential Workflow Pipeline

              Execution map and session handoff order

              Step 0

              Document Parsing

              MCP extracts plain text sections from the uploaded PDF resume file.

              Step 1

              Structured Resume Analysis

              Candidate profile is extracted via structured outputs (`ResumeAnalysisOutput`).

              Step 2

              Job Deconstruction

              Target job description requirements are modeled into Pydantic models.

              Step 3

              Gap and Overlap Analysis

              Overall match score and specific skill discrepancies are calculated.

              Step 4

              Personalized Study Timeline

              A structured 30/60/90 day action plan and portfolio projects are formulated.

              Step 5

              Tailored Interview Simulator

              Technical questions and expected behavioral templates are curated.

              Step 6

              Synthesizer Agent

              Outputs final Career Intelligence Report to user.
